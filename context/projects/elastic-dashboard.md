# Elastic — Dashboard

**Status:** Active — stalled on client, now potentially unblocked (May 21)
**Local path:** ./elastic-dashboard
**Stack:** Vite, Vue.js

## Background
C-Suite metrics dashboard for Elastic (the company). Tracks PR/comms metrics across three columns: Visibility (SoM, T1 placement, LLM citations TBD), Credibility (message pull-through, signals of authority, trust), Social (engagement rate). Board Metrics section at top TBD. Data pulled from Delve API.
